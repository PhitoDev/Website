# Professional Website Rewrite 
This is my old website and blog that I would like to update for 2026. I would like to keep the content and change frameworks and update the UI/UX.
---
## Old Tech Stack
The old project tech stack is essentially as follows:
* Nuxt 3
* Typescript 
* Netlify CMS
* Nuxt Content
* Bulma CSS
* Tailwind CSS

The more detailed specifics are available in the different configuration files.

## New Tech Stach
The rewrite will use the latest of the following frameworks and libraries:
* Next.js
* Typescript 
* Decap CMS 
* Material UI React component 
* Markdown-to-jsx

Reuse the updated dependencies from the old configuration wherever appropriate.

## Guidelines 
* Start with creating a backup of this project for reference 
* Maintain a similar project structure but prefer Next best practices where they differ from Nuxt.
* Update stylins, typography and color scheme to be in line with Material 3 guidelines.
* Revamp UI/UX to a clean and modern feel. Focus on accessibility and responsive design.
